# ebook-local-web

로컬에서 실행하는 웹 버전입니다.

## 실행

```bash
npm install
npm run start
```

브라우저에서 `http://localhost:3199` 접속.

## 현재 구현

- 서울 전역 전자도서관(구 단위 포함) 병렬 검색
- 책 제목 기준 후보 파싱 및 상태 추정
  - 소장권수
  - 대출가능권수
  - 대출중권수
  - 예약/대기권수
- 예약 오판 방지 규칙 반영
  - `소장 1` + `예약 > 0` + `명시적 대출가능 없음` => 예약 상태 우선
- 흐름 우선순위
  1. 서울 전역 전자도서관 대출 가능 확인
  2. 없으면 은평구 통합검색 링크 제안
  3. 그래도 없으면 교보 SAM 링크 제안
- 로그인 보조
  - 각 도서관별 로그인 페이지 바로가기 제공

## 주의

- 교보/도서관 페이지 구조가 자주 바뀌므로 파싱 신뢰도는 사이트별로 달라질 수 있습니다.
- 실제 대출/예약 확정은 각 도서관 상세 페이지에서 최종 확인하세요.
